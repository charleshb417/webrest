Inside Baltimore

Inside Baltimore allows users to visualize different datasets that are provided by Baltimore City via Open Baltimore. This application was built primarily using Spring, Hibernate, AngularJS, and D3JS.
Currently, the application contains two datasets:
  • Vacant housing data from 2011 - 2016
  • Crime data from January through November 2016
Currently, users can navigate this data as follows:
  • The Adjacency Matrix creates a heatmap grid to visualize how often two data elements occurred. For example, if there were a large amount of burglaries in the Federal Hill neighborhood, the intersection of those two values would be more intense then other cells. Users can select which keys to compare by setting the "X-Axis" and "Y-Axis" dropdown lists to the appropriate key. This chart is accessible by clicking "Select Dashboard" and selecting "Adjacency Matrix."
  • The Frequency Chart sizes bubbles based on how many occurrences of a value is seen in the data. Users can select what key to sort on from the "Frequency Key" dropdown list. This chart is accessible by clicking "Select Dashboard" and selecting "Frequency Chart".
When viewing a visualization, click on any item to view the data related to that item.
